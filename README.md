# 基于 Spark 的城市居民慢病健康体征多维分析与风险评估系统
——以糖尿病健康指标分析为例

## 项目背景
本项目旨在构建一套针对城市居民健康数据的多维分析与风险评估体系。通过大数据技术，实现从数据存储、多维特征关联分析到慢病风险预测的完整闭环，为辅助临床评估提供数字化参考。

## 系统架构
- **数据层 (Data Lake)**: 基于 **HDFS** 进行大规模健康调查数据 (BRFSS) 的持久化存储。
- **分析层 (OLAP)**: 采用 **Hive** 实现数据的多维查询与指标清洗，完成初步的统计探索。
- **计算层 (Distributed ML)**: 使用 **Spark MLlib** 驱动分布式模型训练，对比逻辑回归与随机森林的预测性能。
- **展示层 (Visualization)**: 基于 Python 多维可视化工具，直观展现慢病指标的特征相关性与人群分布差异。

## 系统功能模块
1. **多维数据分析**: 通过相关性热力图（Correlation Heatmap）识别高风险健康指标，揭示指标间的共线性关系。
2. **风险评估模型**: 结合特征工程优化，构建基于决策树集成与线性回归的慢病风险评估算法。
3. **特征可视化**: 针对核心指标（如 BMI 等）进行分布分析，对比患病人群与健康人群的体征差异。

## 系统分析结论
* **相关性发现**: 实验验证了 BMI、HighBP 等指标与糖尿病风险存在显著强相关，通过多维分析成功剔除了冗余指标（MentHlth, PhysHlth）。
* **模型评估**: 系统支持逻辑回归与随机森林模型对比。实验表明，逻辑回归在保持 0.8240 AUC 指标的同时，提供了更强的解释性，更适用于医疗辅助评估场景。

## 文件说明

- `diabetes_analysis.py`: 多维特征关联分析脚本。
- `correlation_heatmap.png`: 多维体征相关性热力图。
- `bmi_distribution.png`: 慢病风险人群体征分布对比图。
