# 基于 Spark 的城市居民慢病风险评估系统 —— 以糖尿病健康指标分析为例

## 项目简介

本项目基于 Hadoop 与 Spark 大数据生态，针对 BRFSS 居民健康调查数据集，实现了从 HDFS 数据存储、Hive 数据探索到 Spark MLlib 分布式机器学习建模的完整开发流程。

系统通过分析 25 万条居民健康数据，研究 BMI、吸烟、高血压等健康指标对糖尿病风险的影响，并构建糖尿病风险评估模型。

---

## 项目背景

随着慢病防治工作的推进，利用大数据技术进行居民健康风险预测已成为智慧医疗的重要方向。

本项目基于 Spark MLlib 构建分布式机器学习模型，对居民慢病风险进行预测与评估，验证大数据技术在医疗健康场景中的应用价值。

---

## 技术栈

* 开发环境：Ubuntu 伪分布式环境

## 系统功能模块

### 1. 数据分层治理（Data Pipeline）

#### ODS 层（原始数据层）

* 存储原始 BRFSS 健康调查数据
* 数据上传至 HDFS 统一管理

#### DWD 层（数据明细层）

* 完成缺失值处理
* 数据清洗
* 异常值过滤

#### DWS 层（数据服务层）

* 特征向量化
* 构建机器学习输入数据
* 输出模型训练数据集

---

### 2. 多维特征关联分析

系统通过 Spark SQL 与 Python 可视化工具，对居民健康指标进行相关性分析：

* BMI 与糖尿病风险呈显著正相关
* 高血压是重要风险因素
* MentHlth 与 PhysHlth 存在较强共线性
* 剔除冗余特征提升模型效率

---

### 3. 多元化风险评估模型

系统基于 Spark MLlib 实现两种机器学习模型：

#### （1）逻辑回归模型

适用于二分类风险预测，具有良好的可解释性。

#### （2）随机森林模型

利用多棵决策树进行集成学习，提高模型泛化能力。

---

### 4. 模型优化策略

项目采用：

* 5 折交叉验证（Cross Validation）
* 参数网格搜索（ParamGridBuilder）
* AUC 指标评估模型性能

对随机森林与逻辑回归模型进行参数调优。

## 运行说明

### 1. 启动 Hadoop 与 Spark 环境

```bash
start-all.sh
```

---

### 2. 数据清洗与分析

运行特征分析脚本：

```bash
python3 scripts/diabetes_analysis.py
```

生成内容包括：
* correlation_heatmap.png
* bmi_distribution.png
---

### 3. Spark 分布式模型训练

提交 Spark 任务：

```bash
spark-submit scripts/diabetes_pipeline_rf.py
```

系统将自动完成：

* 数据读取
* 特征工程
* 模型训练
* 模型评估
* AUC 输出

## 实验结果

### 1. 特征分析结果

通过相关性热力图发现：

* BMI 与糖尿病风险具有显著相关性
* 高血压与高血糖人群关联明显
* 不良生活习惯会增加慢病风险

---

### 2. 模型性能结果

| 模型   | AUC   |
| ---- | ----- |
| 逻辑回归 | 0.81+ |
| 随机森林 | 0.82+ |

实验结果表明：

* Spark MLlib 能够高效处理大规模健康数据
* 随机森林模型在风险预测任务中表现更优
* 系统满足基础临床辅助评估需求
